## Installation

```
git clone https://github.com/thiagobraga/dotfiles
cd dotfiles
. ./install
```

## Contents

- Terminator configurations
- Vi configurations
- zsh configurations with Spaceship theme configured
- Composer on PATH
- npm on PATH
- "Windows" shortcut to restart in Windows for dual-boot systems
